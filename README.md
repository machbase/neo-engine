# neo-engine

Go binding for Machbase time-series database core.

## Install

```sh
go get -u github.com/machbase/neo-engine
```

## Supporting platforms

| OS       | ARCH          | SUPPORT     |
|:---------|:--------------|:-----------:|
| Linux    | amd64         | O           |
|          | arm64         | O           |
|          | arm (32bit)   | O           |
| macOS    | amd64 (Intel) | O           |
|          | arm64 (Apple) | O           |
| Windows  | amd64         | O           |
